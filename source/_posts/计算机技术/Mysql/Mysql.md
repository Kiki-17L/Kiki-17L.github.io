# 一、数据类型


## 1. 数值类型



### 1.1 严格数值类型

| 类型           | 大小                                     | 范围            |
| -------------- | ---------------------------------------- | --------------- |
| TINYINT        | 1B                                       | 0-255           |
| SMALLINT       | 2B                                       | 0-6 5535        |
| MEDIUMINT      | 3B                                       | 0-16 777 215    |
| INT（INTEGER） | 4B                                       | 0-4 294 967 295 |
| BIGINT         | 8B                                       |                 |
| DEC（DECIMAL） | 对DECIMAL(M,D) ，如果M>D，为M+2否则为D+2 |                 |



### 1.2 近似数值类型

| 类型   | 大小 | 范围 |
| ------ | ---- | ---- |
| FLOAT  | 4B   |      |
| DOUBLE | 8B   |      |



## 2. 日期和时间类型

| 类型      | 大小 | 范围                                                   | 格式                |
| --------- | ---- | ------------------------------------------------------ | ------------------- |
| DATE      | 3B   | 1000-01-01/9999-12-31                                  | YYYY-MM-DD          |
| TIME      | 3B   | '-838:59:59'/'838:59:59'                               | HH:MM:SS            |
| YEAR      | 1B   | 1901/2155                                              | YYYY                |
| DATETIME  | 8B   | '1000-01-01 00:00:00' 到 '9999-12-31 23:59:59'         | YYYY-MM-DD hh:mm:ss |
| TIMESTAMP | 4B   | '1970-01-01 00:00:01' UTC 到 '2038-01-19 03:14:07' UTC | YYYY-MM-DD hh:mm:ss |



## 3. 字符串类型

| 类型       | 大小           | 用途                            |
| ---------- | -------------- | ------------------------------- |
| CHAR       | 0-255 B        | 定长字符串                      |
| VARCHAR    | 0-65535 B      | 变长字符串                      |
| TINYBLOB   | 0-255 B        | 不超过 255 个字符的二进制字符串 |
| TINYTEXT   | 0-255 B        | 短文本字符串                    |
| BLOB       | 0-65535 B      | 二进制形式的长文本数据          |
| TEXT       | 0-65535 B      | 长文本数据                      |
| MEDIUMBLOB | 0-65535 B      | 二进制形式的中等文本数据        |
| MEDIUMTEXT | 0-65535 B      | 中等文本数据                    |
| LONGBLOB   | 0-16 777 215 B | 二进制形式的极大文本数据        |
| LONGTEXT   | 0-16 777 215 B | 极大文本数据                    |
