一、数据类型

数据类型 描述 阈值
String 基本类型，二进制安全的字符串 最大 512MB
Hash String类型的键值对集合，存储对象 \(2^{32}-1\) 个键值对
List String类型的列表，按插入顺序排序 \(2^{32}-1\) 个元素
Set String类型的无序集合 \(2^{32}-1\) 个成员
Zset String类型的有序集合 \(2^{32}-1\) 个成员
